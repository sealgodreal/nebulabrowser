PRIVACY POLICY (updated 4/10/26):
nebula does not collect or store any sort of personal data.
web requests are routed through the proxy server to load external content and will most likely never leave a trace of your data.
our proxy works by receiving the request the user sends, then it uses the backend to externally load the targeted service. all the data that the service sees is an automated request sent from the proxy ip.
**we absolutely do not distribute or sell any sort of data.**
